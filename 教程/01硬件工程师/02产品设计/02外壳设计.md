# 外壳设计

我参考的[是这一个视频](https://www.bilibili.com/list/watchlater?oid=946837767&bvid=BV1DW4y1K7Ww&spm_id_from=333.788.top_right_bar_window_view_later.content.click), 我的设计目标是一个可以放在桌上的天气时钟

在实际设计的时候, 需要注意的点是

1. 设计的时候有一定的仰角方便观看
2. 有一个屏幕支持
3. 预留USB线的位置
4. 留出来按键的位置
